# Roadmap

## Near term

- Add screenshot assets for README and social previews.
- Add per-run detail pages with event-level drilldown.
- Add exportable markdown/HTML run reports.
- Add configurable thresholds for drift, failures, and retries.

## Medium term

- Add optional SQLite storage for larger local datasets.
- Add session-level grouping across multiple related runs.
- Add anomaly detection based on deterministic rolling windows.
- Add schema examples for tool-level events and memory-level events.

## Long term

- Support OpenTelemetry-style trace ingestion.
- Add integrations for local agent frameworks while keeping API calls optional.
- Add benchmark comparisons across agent versions.
